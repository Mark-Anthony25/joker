## Usage

Simply open `index.html` in your web browser or visit the hosted website to start generating jokes!

### Running Locally

1. Clone this repository
2. Open `index.html` in your web browser
3. Click the "Generate Joke" button to get a new joke

No server or installation required!

## API

This project uses [JokeAPI](https://jokeapi.dev) - a free REST API that serves uniformly and well-formatted jokes.
